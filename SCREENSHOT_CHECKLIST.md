## 📋 Image Specifications

**All Images Should Be:**
- ✅ Width: 1200px (minimum) or 1920px (preferred)
- ✅ Format: PNG (preferred) or JPEG
- ✅ File size: Under 500KB
- ✅ Color space: sRGB
- ✅ Compressed using TinyPNG or ImageOptim

**Privacy & Security Requirements:**
- ❌ NO real personal data, names, or ID numbers
- ❌ NO unblurred faces in KhunCaddy verification images
- ✅ Use sample/demo data or fake names
- ✅ Anonymize or redact sensitive information
- ✅ Use staging environments when possible

## 💡 Pro Tips for Screenshots

1. **Use Staging/Demo Environments**
   - Never screenshot production with real customer data
   - Use test accounts and sample data instead

2. **Make Them Look Professional**
   - Clear, clean UI without clutter
   - Good lighting for photos (if taking physical screenshots)
   - Zoom in appropriately (not too zoomed in or out)

3. **Highlight Key Features**
   - For ChatRecept: Show AI helpfulness, conversation flow
   - For NexaMatrix: Make Attendance Heatmap vibrant and eye-catching
   - For KhunCaddy: Emphasize verified badges and trust indicators

4. **Consistent Style**
   - Try to capture all images in similar lighting/conditions
   - Use same browser/device type where possible
   - Maintain consistent color grading

5. **Test Responsiveness**
   - Capture at: 375px, 768px, 1024px, 1920px widths
   - Ensure images look good at all sizes

## 🔒 Privacy Checklist

Before uploading any screenshot:
- [ ] No real customer names or personal data visible
- [ ] No real phone numbers, emails, or IDs visible
- [ ] No real ID documents or sensitive documents visible
- [ ] All faces in KhunCaddy verification are blurred/redacted
- [ ] Company financial data is anonymized or uses samples
- [ ] Location data is generic (e.g., "Bangkok" not real addresses)
