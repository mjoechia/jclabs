### File Format
- **Format:** JPG (JPEG) for photos, PNG for UI screenshots with transparency needed

- **Quality:** High quality, professional appearance

### File Size
- **Splash images:** <400KB each
- **Detail images:** <500KB each
- **Mobile images:** <300KB each
- **Total budget:** ~12MB for all 24 images

- Use TinyPNG (https://tinypng.com) or ImageOptim for compression

- ✅ Anonymize employee/user data

- ✅ Use staging environments, not production
